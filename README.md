# GIFmything
Guess the phrases using only GIFs.

## Instructions:
1. Update your npm to the most recent version.
```
sudo npm upgrade npm -g
```
2. Clone the repository.
```
git clone https://github.com/greenteas/GIFmything.git
```

3. Install dependencies.
```
npm install
```

4. Run the game!
```
node server.js
```
